Hadas - an anti-IDE

An open source mini-server that continuously recycles your (node.js) project as it changes.

Download this project as a .zip file Download this project as a tar.gz file

Get immediate feedback as you develop your project, by letting your runtime mirror your code as it evolves. It recycles your server side and any open browser window connected to it, once your code changes. Makes you more efficient.

An html user interface governs enabling and disabling, as well as showing whether the project is running and serving its node console output, all in one place. You can even share the console with your peers.

This is a hacked-together 'working on my machine' project, but if you're a hacker, head out to the Github repo for grabbing the code and checking out setup complexity.
